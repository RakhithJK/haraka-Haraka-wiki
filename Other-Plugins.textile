h2. Haraka Plugins

* "Haraka-CouchDB":http://github.com/maxogden/haraka-couchdb - Store incoming mail in CouchDB
* "Haraka API Server":https://github.com/KellyLSB/Haraka-API-Server - Give your Haraka an outgoing API access comparable to PostmarkApp over HTTP POST.
* "Haraka-HTTP-Forward":https://github.com/jplock/haraka-http-forward - Forward incoming mail to a remote HTTP server.
* "Haraka LDAP":https://github.com/tasansga/haraka-ldap - various LDAP integrations
